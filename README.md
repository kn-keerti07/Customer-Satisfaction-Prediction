# 🏦 AI-Based Customer Satisfaction Prediction in Banking

## 📌 Project Overview

This project predicts customer satisfaction in the banking sector using Machine Learning techniques. Customer churn is considered an indicator of dissatisfaction. The system analyzes customer financial and behavioral data to identify customers who are likely to leave the bank.

The project helps banks:

* Improve customer experience
* Reduce customer churn
* Enhance decision-making
* Improve service quality

## 📂 Dataset

**Dataset Name:** Churn_Modelling.csv

The dataset contains customer information such as:

* Credit Score
* Geography
* Gender
* Age
* Tenure
* Account Balance
* Number of Products
* Credit Card Status
* Active Member Status
* Estimated Salary

**Target Variable:**

* Exited

  * 0 = Customer Retained (Satisfied)
  * 1 = Customer Left (Dissatisfied)
